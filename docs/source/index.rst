AgentGuard
==========

Inter-agent security firewall for multi-agent AI systems
(LangGraph, CrewAI, AutoGen).

**PyPI:** ``pip install inter-agent-guard`` — **import:** ``agentguard``

.. code-block:: bash

   pip install "inter-agent-guard[all,otel]"
   python scripts/download_release_model.py
   agentguard status
